# Todo Management System

A professional and user-friendly Todo Management System developed using Node.js, Express.js, and EJS. This application allows users to efficiently manage daily tasks with features like adding, updating, deleting, and tracking task status through a clean brown-themed responsive interface.

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Track task progress
- Task priority management
- Responsive brown-themed UI
- Dynamic rendering using EJS
- Simple and clean user experience

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS
- HTML5
- CSS3

## 📌 Functionalities

### Dashboard

Displays all tasks with:
- Task ID
- Title
- Description
- Priority
- Status

### Add Task

Users can create new tasks with:
- Title
- Description
- Priority
- Status

### Edit Task

Allows updating:
- Task details
- Priority
- Current status

### Delete Task

Removes tasks permanently from the list.
